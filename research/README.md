# リサーチフォルダ

半導体ニュース記事作成前のリサーチ情報を保存するフォルダです。

## フォルダ構成
research/
├── YYYY-MM/
│   ├── YYYYMMDD-テーマ名.md
│   └── ...
└── template.md

## 運用ルール
- ファイル名：YYYYMMDD-テーマ名.md（例：20260502-dram-hbm.md）
- リサーチ完了後、記事作成フェーズに移行
- 記事公開後も削除せず保存する
- 公式発表・一次情報を優先する
- ニュース本文の転載は禁止
- 記事本文は独自解説として作成する

## 著作権ルール
- ニュース本文の転載は禁止
- 引用は最小限にする（15ワード以内）
- 基本は公式発表・一次情報を優先
- 記事本文は独自解説として作成する

## ステータス管理
- research：リサーチ中
- draft：記事下書き作成済み
- ready：確認済み・投稿待ち
- posted：WordPress投稿済み
