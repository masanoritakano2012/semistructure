# 記事フォルダ

research/ のリサーチファイルをもとに作成した記事を保存します。

## フォルダ構成
articles/
├── draft/   # AIが生成した未確認記事
├── ready/   # 人間が確認済みの記事
└── posted/  # WordPress投稿済みの記事

## 運用ルール
- WordPressへ投稿する前に、必ずready/へ移動する
- WordPress投稿ステータスは必ずdraft（下書き）にする
- 公開はWordPress管理画面から手動で行う
- アイキャッチ画像は共通テンプレートを使用する
